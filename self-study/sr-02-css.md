# Самостійна робота №2: CSS та сучасні підходи до стилізації

**Тема:** CSS та сучасні підходи до стилізації

**Мета:** Вивчити сучасні можливості CSS, опанувати CSS Grid та Flexbox для побудови макетів, навчитися використовувати CSS Variables та препроцесор SASS/SCSS.

**Кількість годин:** 4

## Теоретичний матеріал

### Flexbox

Flexbox (Flexible Box Layout) - модель розташування елементів в одному напрямку (рядок або стовпець). Контейнер отримує `display: flex`, після чого дочірні елементи стають flex-елементами. Основні властивості контейнера: `flex-direction` (напрямок), `justify-content` (вирівнювання по головній осі), `align-items` (вирівнювання по поперечній осі), `flex-wrap` (перенос рядків), `gap` (відстань між елементами). Властивості елементів: `flex-grow`, `flex-shrink`, `flex-basis`, скорочення `flex`, `align-self`, `order`.

### CSS Grid

CSS Grid - двовимірна система розташування елементів (рядки та стовпці одночасно). Контейнер отримує `display: grid`. Основні властивості: `grid-template-columns` та `grid-template-rows` (визначення сітки), `gap` (відстані), `grid-template-areas` (іменовані зони). Функція `repeat()` спрощує повторення, `fr` - гнучка одиниця виміру, `minmax()` - діапазон розмірів. Елементи розміщуються через `grid-column`, `grid-row` або `grid-area`.

### CSS Variables (Custom Properties)

CSS Variables дозволяють визначати значення, які повторно використовуються у стилях. Оголошуються через `--назва: значення` та застосовуються через `var(--назва)`. Зазвичай визначаються у `:root` для глобальної доступності. Підтримують каскадність - можна перевизначити для окремих елементів. Ідеальні для створення тем (темна/світла тема).

### SASS/SCSS

SASS - CSS-препроцесор, який розширює можливості CSS. SCSS - синтаксис SASS, сумісний з CSS. Основні можливості: змінні (`$color: #333`), вкладеність селекторів, міксини (`@mixin`, `@include`), функції, часткові файли (`_variables.scss`) та імпорт (`@use`), оператори та умовні конструкції.

### Адаптивний дизайн

Media queries (`@media`) дозволяють застосовувати стилі залежно від розміру екрану. Підхід Mobile First - спочатку стилі для мобільних, потім розширення через `min-width`. Контейнерні запити (`@container`) - новий стандарт для адаптації на рівні компонентів.

## Контрольні питання

1. У чому принципова різниця між Flexbox та CSS Grid? Коли доцільно використовувати кожен з них?
2. Що таке одиниця виміру `fr` у CSS Grid і як вона розподіляє простір?
3. Як працює CSS-властивість `gap` і де вона підтримується?
4. Як реалізувати перемикання між темною та світлою темою за допомогою CSS Variables?
5. Що таке міксини (mixins) у SASS та яку проблему вони вирішують?
6. Чим директива `@use` відрізняється від `@import` у SASS?
7. Що таке підхід Mobile First і чому він рекомендований?
8. Як працюють контейнерні запити (`@container`) і чим вони відрізняються від media queries?

## Практичне завдання

1. Створіть макет dashboard-сторінки за допомогою CSS Grid з бічною панеллю, хедером та основним контентом.
2. Реалізуйте картки товарів у основному контенті за допомогою Flexbox.
3. Визначте CSS Variables для кольорів та розмірів, реалізуйте перемикання теми.
4. Додайте адаптивність: на мобільних пристроях бічна панель має ховатися, а картки - відображатися в одну колонку.
